# Phase 01: Cleanup Old Examples

## Scope

Delete all existing example files and orphaned directories to prepare for the new flat structure.

**What will be built:**
- Clean slate for new examples
- Remove `example/ModernglRender/` directory entirely
- Remove `example/benchmark.py`

**Dependencies:**
- None (first phase)

**Files affected:**
- DELETE: `example/ModernglRender/` (entire directory)
- DELETE: `example/benchmark.py`

## Implementation

### Task 1: Delete ModernglRender directory
**Action:** Remove the entire `example/ModernglRender/` directory including all subdirectories and files.

**Files to delete:**
- `example/ModernglRender/enhanced_camera_demo.py`
- `example/ModernglRender/interactive_interface_demo.py`
- `example/ModernglRender/multiview_example.py`
- `example/ModernglRender/rgba_demo.py`
- `example/ModernglRender/v031_features_demo.py`
- `example/ModernglRender/pyvr/` (orphaned directory)
- `example/ModernglRender/tests/` (orphaned directory)
- `example/ModernglRender/__pycache__/`
- `example/ModernglRender/` (directory itself)

**Command:**
```bash
rm -rf example/ModernglRender/
```

### Task 2: Delete old benchmark.py
**Action:** Remove the old benchmark script from example root.

**File to delete:**
- `example/benchmark.py`

**Command:**
```bash
rm example/benchmark.py
```

### Task 3: Verify clean state
**Action:** Confirm the example directory is empty and ready for new files.

**Command:**
```bash
ls -la example/
```

**Expected result:**
- Directory exists but contains no Python files
- May contain `.DS_Store` or similar OS files (acceptable)

## Verification

### Step 1: Verify directory structure
```bash
# Check example directory contents
ls -la example/

# Should show empty directory (or only OS files like .DS_Store)
```

### Step 2: Verify no Python files remain
```bash
# Search for any remaining Python files
find example/ -name "*.py"

# Should return no results
```

### Step 3: Verify orphaned directories are gone
```bash
# These should not exist
ls example/ModernglRender/pyvr/
ls example/ModernglRender/tests/

# Should return "No such file or directory"
```
